# ⏱️ Time Series Forecasting

> Pronosticar la demanda de energía eléctrica a partir de datos históricos horarios.

**Estado:** 🚧 En construcción — ver [`ROADMAP.md`](ROADMAP.md) para el plan completo por fases.

## 🎯 Objetivo
Análisis de estacionalidad y tendencia, descomposición de la serie y **forecasting con ARIMA y
Prophet** (más un baseline estacional) sobre el consumo eléctrico horario de PJM.

## 🛠️ Stack previsto
Python · pandas · statsmodels · Prophet · scikit-learn · matplotlib / plotly

## 📄 Dataset
[Hourly Energy Consumption — Kaggle](https://www.kaggle.com/datasets/robikscube/hourly-energy-consumption)
(serie usada: `PJME_hourly.csv`; no incluido en el repo — ver instrucciones en el ROADMAP).
